Serverless Lead Capture Form

Integration Example

This page demonstrates how the lead capture form integrates seamlessly into any website. The form adapts to your site's design while maintaining its functionality and accessibility.

Key Features

  • Real-time Validation: Email addresses and other fields are validated as users type
  • Responsive Design: Works perfectly on desktop, tablet, and mobile devices
  • Theme Support: Light and dark themes that adapt to your site
  • Accessibility: Full keyboard navigation and screen reader support
  • No Dependencies: Pure vanilla JavaScript with no external libraries

How It Works

Simply add a div with the appropriate data attributes and include our JavaScript file. The form will automatically initialize and handle all validation, submission, and user feedback.

Quick Integration

Add this to your HTML:

<div id="lead-capture-form" data-api-endpoint="https://your-api.com/leads" data-fields="name,email,company"> </div> <script src="lead-capture.js"></script>

Configuration Options

The form supports various configuration options through data attributes:

Available Fields

  • name - Full name (required by default)
  • email - Email address (always required)
  • company - Company name (optional)
  • phone - Phone number (optional)
  • details - Additional details text (optional)

Configuration Attributes

  • data-api-endpoint - Your API endpoint URL (required)
  • data-fields - Comma-separated list of fields to include
  • data-title - Form title text
  • data-submit-text - Submit button text
  • data-success-message - Success message text
  • data-error-message - Error message text
  • data-theme - "light" or "dark"
  • data-width - "full" or "compact"

Example Configuration

<div id="lead-capture-form" data-api-endpoint="https://api.yourdomain.com/leads" data-fields="name,email,company,phone,details" data-title="Contact Our Sales Team" data-submit-text="Request Quote" data-success-message="Thank you! We'll send you a quote within 2 hours." data-theme="light" data-width="full"> </div>

Real-time Validation Examples

Try these examples to see the validation in action:

Email Validation

  • Try: invalid-email (missing @ and domain)
  • Try: test@ (missing domain)
  • Try: @domain.com (missing username)
  • Valid: user@example.com

Name Validation

  • Try: A (too short, minimum 2 characters)
  • Try: John123 (numbers not allowed)
  • Valid: John Doe, Mary O'Connor, Jean-Pierre

Phone Validation

  • Try: abc123 (letters not allowed)
  • Valid: +1234567890, 555-123-4567, (555) 123-4567
